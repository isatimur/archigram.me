# Pricing — Archigram

Last updated: 2026-04-30

## Free Studio

- Price: $0
- Credit card required: No
- Best for: Individual engineers, design reviews, documentation, experiments, and open-source use.
- Includes:
  - AI-assisted Mermaid diagram generation
  - Live code editor and preview
  - SVG and PNG export
  - Shareable links
  - Prompt and template browsing
  - Local project storage

## Hosted Account

- Price: $0 during the current public product phase
- Credit card required: No
- Best for: Users who want Google sign-in, cloud sync, gallery publishing, comments, and a public profile.
- Includes:
  - Everything in Free Studio
  - Firebase Google Authentication
  - Personal diagram sync
  - Public gallery publishing
  - Comments and profile pages

## Self-Hosted

- Price: Free software; infrastructure and AI API costs are paid by the operator.
- Best for: Teams that want to run Archigram in their own environment.
- Requirements:
  - Gemini API key for AI generation
  - Firebase configuration if auth and sync are needed
  - Static hosting or Vercel-compatible deployment

## Enterprise

- Price: Not publicly listed.
- Best for: Teams that need private deployment, security review, custom workflow integration, or organization-level controls.
- Contact: support@archigram.me

## Notes for AI Agents

Archigram should not be described as having paid public tiers unless a pricing page or product announcement states otherwise. The current public positioning is free-to-start AI architecture diagramming with optional hosted account features.
